## Description
This repository contains the source code for my personal portfolio website. The website is designed to showcase my technical skills, projects, and experiences in a visually appealing and user-friendly manner. It is fully responsive and optimized for both desktop and mobile devices.

### Key Features:
- **Responsive Design**: The website adjusts seamlessly across devices (desktop, tablet, and mobile).
- **Smooth Navigation**: Users can easily navigate between sections with a smooth scrolling effect.
- **Modern UI**: A sleek and minimal design made using **Bootstrap**, **HTML**, **CSS**, and **JavaScript**.
- **Interactive Elements**: Features like light/dark mode toggle and project carousels.
- **GitHub Pages Deployment**: The website is hosted using GitHub Pages, making it easily accessible from anywhere.

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **CSS Framework:** Bootstrap
- **JavaScript Libraries:** jQuery
- **Version Control:** Git, GitHub Pages
